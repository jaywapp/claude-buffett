## GitHub 이슈로 기업 추가 요청하는 방법

GitHub 이슈([jaywapp/claude-buffett](https://github.com/jaywapp/claude-buffett/issues))에 아래 형식으로 등록하면 매일 오전 4시에 자동 반영됩니다.

**라벨:** `coverage-add`

**이슈 제목 형식:**
```
[기업추가] 기업명
```
티커 번호는 몰라도 됩니다. 메시가 자동으로 조회합니다.

**이슈 본문 (선택):**
```
섹터: 반도체/AI
사유: 설명 (생략 가능)
```

**예시:**
- 제목: `[기업추가] 카카오`
- 제목: `[기업추가] 두산에너빌리티`
